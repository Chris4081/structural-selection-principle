# Paper 37 — MAAT v1.2.1 Observable Proxy Predictions and Stability Landscape

This folder contains the reproducibility material for:

**MAAT v1.2.1 Observable Proxy Predictions and Stability Landscape**  
Structural Fields, Projection Layer, CCI Diagnostics, and Parameter Sensitivity

The experiment is the numerical companion to the v1.2.1 conceptual closure.
It uses four primary support sectors `H, B, S, V`, derives structural respect
as `R_resp = (H B V)^(1/3)`, and defines emergent robustness as
`R_rob = min(R_resp, (H B S V)^(1/4))`.

## Scripts

| Script | Role |
|--------|------|
| `paper37_observables_emergent_robustness.py` | Generates the baseline v1.2.1 observable proxy trajectory and figures. |
| `paper37_stability_landscape.py` | Scans the two-parameter proxy landscape over projection strength and damping. |
| `sat_validation/maat_v121_sat_validation.py` | Reproduces the SAT correlation table used in the empirical discussion. |

## Reproduce

From this folder, run:

```bash
python3 paper37_observables_emergent_robustness.py
python3 paper37_stability_landscape.py
cd sat_validation
python3 maat_v121_sat_validation.py maat_cosmos_full_results.csv
```

The scripts write outputs to:

| Output folder | Contents |
|---------------|----------|
| `observable_outputs/` | baseline CSV/JSON output and observable figures |
| `stability_landscape_outputs/` | scan CSV/JSON output and heatmap figures |
| `sat_validation/` | SAT input table, v1.2.1 validation output, correlations, and summaries |

## Scientific Status

This is a toy/proxy benchmark, not an observational cosmology fit.
The growth deviations are simplified proxy quantities and are not
Boltzmann-code predictions.
