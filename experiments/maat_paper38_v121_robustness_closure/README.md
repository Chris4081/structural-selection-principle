# Paper 38 — MAAT v1.2.1 Robustness Closure in Linear Growth

This folder contains the reproducibility material for:

**MAAT v1.2.1 Robustness Closure in Linear Growth**  
Sub-Percent Growth Signatures, Emergent Robustness, and Selection-Field Stability

The experiment applies the MAAT v1.2.1 closure

```text
R_resp = (H B V)^(1/3)
R_rob  = min(R_resp, (H B S V)^(1/4))
Stability = R_rob
```

to a minimal linear-growth proxy and to small selection-field perturbations.

## Script

| Script | Role |
|--------|------|
| `maat_paper38_v121_robustness_closure.py` | Reproduces the growth comparison, v1.2.1 closure diagnostics, selection-field perturbation test, positivity test, and summary outputs. |

## Reproduce

From this folder, run:

```bash
python3 maat_paper38_v121_robustness_closure.py
```

The script writes outputs to:

| Output | Contents |
|--------|----------|
| `paper38_v121_outputs/paper38_summary.json` | Key numerical summary |
| `paper38_v121_outputs/paper38_growth_curves.csv` | Linear growth and `f sigma_8` proxy curves |
| `paper38_v121_outputs/paper38_v121_structural_closure.csv` | H/B/S/V, `R_resp`, `R_rob`, and CCI diagnostics |
| `paper38_v121_outputs/paper38_dlambda_evolution.csv` | Selection-field perturbation trajectories |
| `paper38_v121_outputs/*.png` | Figures used in the manuscript |

## Scientific Status

This is an effective-theory consistency benchmark. It is not a full
Boltzmann-code calculation, not a cosmological likelihood, and not an
observational detection of MAAT. The compact `f sigma_8` points in the figure
are used only as a visual comparison scale.
